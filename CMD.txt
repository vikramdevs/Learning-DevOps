df = disk free Shows disk space usage (storage). df -h

free = Shows RAM and swap memory usage. free -h

nproc = Shows the number of CPU cores available to the system.

top = Shows real-time system performance.

| Command | Checks             | What it shows                             |
| ------- | ------------------ | ----------------------------------------- |
| `df`    | Disk               | HDD / SSD usage (total, used, available)  |
| `free`  | Memory             | RAM and Swap usage                        |
| `nproc` | CPU                | Number of available CPU cores             |
| `top`   | System performance | Live CPU, memory usage, running processes |



set -x = debug mode for shell scripts.



ps -ef = show every running process with full details.

It’s a snapshot of what’s running right now on the system.

ps → process status

-e → all processes (system + users)

-f → full format (extra columns)

to search for a process use and grep is a search command
ps -ef | grep "process_name" = to find a specific process in the list of running processes. eg. ps-ef | grep "nginx"
