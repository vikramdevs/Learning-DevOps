What is DevOps?

DevOps is a collab between Development  and Operations  team that uses automation to deliver software faster and more reliably.

-> Automation
-> Quality
-> Monitoring
-> Testing


Why DevOps?

DevOps is used to deliver software faster, reduce errors, and improve collaboration between teams.
(DevOps isliye use hota hai taaki software fast deliver ho, errors kam ho, aur development aur operations team milkar smoothly kaam kare.)


What is SDLC? (Software Development Life Cycle)

SDLC is a step-by-step process used to plan, develop, test, deploy, and maintain software.


What is JIRA??

JIRA is a project management and issue-tracking tool used to plan, track, and manage software work.


What is Virtual Machine


A Virtual Machine is a software-based computer that runs on a physical machine and works like a real computer, with its own operating system, CPU, memory, and storage.
